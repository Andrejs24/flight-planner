Flight Planner App ✈️
--
Flight Planner is a web application that allows users to plan and manage their flights. With this app, users can add new flights, view existing flights, and search for flights based on different criteria.

Features
--
Add new flights: Users can add new flights to the system by providing flight details such as departure airport, destination airport, carrier, and departure/arrival times.

View existing flights: Users can view a list of all existing flights in the system.

Search for flights: Users can search for flights based on criteria such as departure airport, destination airport, carrier, and departure/arrival times.
